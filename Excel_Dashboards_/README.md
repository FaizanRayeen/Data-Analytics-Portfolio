# 📊 Excel Dashboard Projects

A collection of interactive Excel dashboards built using Pivot Tables, Pivot Charts, and Slicers to analyze real-world business data across e-commerce, customer retention, and retail/F&B domains.

| Project | Domain | Key Highlight |
|---|---|---|
| [🌍 Global E-Commerce Sales Dashboard](./Global-Ecommerce-Sales-Dashboard) | E-Commerce | $438K+ revenue analyzed across regions & products |
| [📉 Customer Churn Analysis Dashboard](./Customer-Churn-Analysis-Dashboard) | Customer Analytics | Churn patterns across 49,999 customers |
| [☕ Coffee Shop Sales Dashboard](./Coffee-Shop-Sales-Dashboard) | Retail / F&B | $698K+ revenue across 3 store locations |

Each project folder contains:
- The full `.xlsx` interactive dashboard file
- A dashboard preview screenshot
- A detailed `README.md` with KPIs, insights, and recommendations

## 🛠️ Tools & Techniques Used

- Pivot Tables & Pivot Charts
- Slicers for interactive filtering
- KPI Cards
- Dashboard layout & design in Excel
